
# Проект создан для тренеровки навыков работы с JSON форматом.

# Для корректной работы проекта необходимо поднять два сервера - отдельно для frontend, отдельно для backend.
#
# Для frontend рекомендуется воспользоваться IDE Visual Studio Code.
# Скачать данную IDE можно на официальном сайте по адресу ниже:

https://code.visualstudio.com/download

# После установки VS Code, необходимо скачать планин - Live Server.
# Плагин доступен для скачивания в Пункте меню "Расширения".
# Так же данное меню можно открыть сочитанием клавиш:

-> CTRL+SHIFT+X

# После установки расширения необходимо открыть HTML файл, который находится по пути:

./frontend/index.html

# В открывшемся файле необходимо выбрать пункт меню - " Open with Live Server "
# Или вопросльзоваться сочетанием клавиш:

-> ATL+L ALT+O

# После открытия файла index.html создаем новое окно терминала, в пункте меню выше - Терминал->Создать терминал. 
# Переходим в директорию backend командой в терминале:

cd backend/

# И запускаем python сервер командой:

python3 main.py

# В директории backend создаются два файла - todos.json и users.json.
# В файле todos.json находятся Задачи, к которым применяем CRUD операции.
