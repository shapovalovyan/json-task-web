# Проект создан для тренеровки навыков работы с JSON форматом.

# А так же в работе с ReactJS + FastAPI приложениями.

# Для работы проекта необходимо скачать Docker.

# В корне проекта запустите команду по запуску двух контейнеров:

docker compose up -d --build

# И проект запуститься по пути:

http://http://localhost/:80

# Порт можно не указывать

# В файле todos.json находятся Задачи, к которым применяем CRUD операции.
